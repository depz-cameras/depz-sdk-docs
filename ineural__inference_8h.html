Detailed Description

Architecture-agnostic orchestration interface for neural depth networks.

Defines DepthInferenceBase (the pipeline-facing ABC) and NeuralInferenceBase (the shared partial implementation) for both stereo and mono depth networks. Mirrors depz_camera_sdk.inference.base.NeuralInferenceBase. Architecture-specific preprocess/postprocess recipes (LightStereo, CREStereo, DepthAnythingV2, Metric3DV2) derive from NeuralInferenceBase in neural_architectures.h. The header is deliberately Python-free so the architecture .cpp files can include it from any translation unit.

Hot-path discipline: submit_sync() runs preprocess then forwards the blobs into the bound InferenceEngine; preprocess/postprocess take cv::Mat (zero-copy from numpy on the binding side).
