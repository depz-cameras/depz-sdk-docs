Decode one raw camera payload into an 8-bit BGR cv::Mat.

Parameters
dataPointer to the source-native wire bytes.
sizeLength of data in bytes.
fourccPixel-format fourcc (uppercase) — "MJPG", "YUYV". Unknown values fall back to cv::imdecode, which succeeds for any container OpenCV recognises (PNG / JPEG / etc.).
widthFrame width in pixels. Used only for raw layouts like YUYV where the wire bytes carry no header. Ignored for compressed formats.
heightFrame height in pixels. Same caveat as width.
outOutput BGR HxWx3 uint8 mat (allocated/resized here).
Returns
true on success; false when decode fails (out.empty() is also true in that case).
