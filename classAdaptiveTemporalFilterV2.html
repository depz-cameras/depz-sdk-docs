Detailed Description

Adaptive Exponential Moving Average (EMA) filter for depth map stabilisation.

A lightweight alternative to the full Kalman filter that adapts the EMA weight \(\alpha\) based on the magnitude of the per-pixel depth change. Large changes (e.g. new object entering the scene) converge immediately; small changes (sensor noise) are suppressed by heavy smoothing.

Per-pixel update rule

\[ \hat{y}_k[u,v] = \alpha[u,v] \cdot z_k[u,v] + \bigl(1 - \alpha[u,v]\bigr) \cdot \hat{y}_{k-1}[u,v] \]

Adaptive blending weight

\[ \alpha[u,v] = \begin{cases} \alpha_{\max} & \text{if } |z_k[u,v] - \hat{y}_{k-1}[u,v]| > \tau \\ \alpha_{\min} & \text{otherwise} \end{cases} \]

where \(\tau\) is the depth-change threshold (in the same units as the input depth map, typically mm for disparity or m for metric depth).

Invalid pixel handling

Pixels with \(z_k = 0\) (unknown depth) are replaced by the last valid estimate \(\hat{y}_{k-1}\) rather than pulling the estimate toward zero.
