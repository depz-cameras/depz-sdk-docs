Detailed Description

Owns the six built-in DepthPipeline filters and applies them per stage.

Holds the speckle, z-clip, edge, temporal-EMA/Kalman, bounding-box, and statistical-outlier filters that run inside the C++ worker loop, matching the Python default_filter_chain(). Params are mutex-protected: setParam() may be called from any thread while the worker runs, and the worker reads a snapshot under the lock once per frame, so changes apply on the next frame without restarting the pipeline.
