Detailed Description

Raw-frame producer behind the DepthPipeline worker loop.

Concrete backends decode nothing — they hand the worker the source-native encoded payload (RawFrame) and let the pipeline run imdecode and the SBS split. Replay sources additionally implement the optional seek extensions.
