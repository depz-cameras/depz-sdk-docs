Detailed Description

Per-network preprocess/postprocess recipes for the SDK's depth models.

One class per neural-depth network, each deriving from NeuralInferenceBase and overriding only preprocess + postprocess (the shared identity / engine glue lives in the base). Parity with the Python reference:

  • LightStereoInferenceCpp — stereo, ImageNet norm, squeeze
  • CREStereoInitialInferenceCpp — stereo, raw 0-255, take ch 0
  • CREStereoCombinedInferenceCpp — stereo, 4 inputs, take ch 0
  • DepthAnythingV2SmallInferenceCpp — mono, ImageNet norm, affine
  • Metric3DV2VitSmallInferenceCpp — mono, CSTM focal recovery, affine

All five build their blobs with cv::dnn::blobFromImageWithParams (the same call the Python reference uses); only postprocess differs per family.
