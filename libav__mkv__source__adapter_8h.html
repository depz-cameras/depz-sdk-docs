Detailed Description

libavformat demuxer behind the IMkvSource interface — the seekable replay head of the worker pipeline.

Feeds the same grab → imdecode → split SBS → rectify → infer → filter → packet chain as the live camera, but from a recorded .mkv, and drives DepthPipeline's replay gate via grab()/seek_us(). One of only two TUs in depz_core that include <libavformat/...>.
