Detailed Description

Mux raw still-encoded frames (the camera's MJPG payload from DepzSource#grab) into a Matroska.

.mkv

that

DepzPipeline.builder().mkvSource(path)

can replay. Minimal video-only writer.

     DepzRecorder rec = DepzRecorder.create("/tmp/clip.mkv", info.width(), info.height(), info.fps(), info.fourcc())) {


    for (...) { RawFrame f = src.grab(1.0); rec.write(f.jpeg(), f.ptsUs()); }
