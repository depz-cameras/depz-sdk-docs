Detailed Description

Pipeline-facing abstract base for a neural depth network.

The only inference surface the pipeline worker sees. Architecture-specific subclasses (LightStereo, CREStereo, …) implement preprocess/postprocess. Inherits IControlOwner so each network declares its tunable surface (empty by default — the built-in backends are control-less).

Provide canvas-pixel intrinsics to focus-aware mono networks.

No-op for stereo and intrinsic-free mono networks; Metric3D V2 / UniDepth override it. The pipeline recomputes fx … cy from the active calibration on every model-resolution change.
