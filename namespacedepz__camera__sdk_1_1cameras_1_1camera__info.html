Detailed Description

Discovery of attached DEPZ cameras.

For each connected video device, queries the server's device-info
endpoint by serial → SKU → :class:`CameraSpec` (via :func:`spec_by_sku`)
and builds a typed :class:`DepzCameraInfo` that carries everything
downstream consumers need: identity (serial, sku, model_name), capability
(``CameraSpec``), and the unit's :class:`Calibration` (already deserialized
from the cereal payload). Cameras whose serial isn't recognized (or
unreachable) are dropped from the list.

The raw device-info dict never leaves this module — the only thing
threaded around is :class:`DepzCameraInfo`.

Network call per attached camera. For a single connected DEPZ camera
this is one round-trip; UI code that surfaces the list paths through
:meth:`list_sources` (web pipeline_builder) which is invoked once per
client connection — acceptable. If you need offline enumeration,
walk the platform backend's :meth:`list_cameras` directly.

Optional[DepzCameraInfo] depz_camera_sdk.cameras.camera_info.get_connected_depz_camera_by_serial ( str  serial) 
Like :func:`list_connected_depz_cameras` but bounded to one serial — runs a
single device-info fetch instead of N.

Used when the caller already knows which camera they want
(``DepthPipeline(source="<serial>")``). Returns ``None`` if no
connected device has that serial, the serial isn't in the server
catalog, or the SKU isn't a registered :class:`CameraSpec`.

◆ get_first_connected_depz_camera()

DepzCameraInfo depz_camera_sdk.cameras.camera_info.get_first_connected_depz_camera ( ) 
Return the first connected DEPZ camera the device-info endpoint
recognizes. Convenience over ``list_connected_depz_cameras()[0]``
for the single-camera quick-start case.

Raises :class:`RuntimeError` when nothing is connected (or no
connected device has a serial registered in the catalog) — clearer
than the ``IndexError`` you'd get from indexing an empty list.

Return every connected video device that the device-info endpoint
recognizes as a registered DEPZ camera class.

:meth:`DepzCameraInfo.from_dict` succeeds.
