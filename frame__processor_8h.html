Detailed Description

Single-call per-frame postprocess chain (disparity→depth→cloud).

Collapses what the Python Postprocessor did across several cross-language trips into one C++ submit. Stateful filters stay in Python; this carries only the math. One FrameProcessor per pipeline, rebuilt on inference/calibration swap.
