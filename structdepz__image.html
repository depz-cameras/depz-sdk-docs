Detailed Description

A borrowed view into one stream's buffer inside a frame.

Points directly at the frame's memory — no copy. Valid only while the owning depz_frame is alive; copy the bytes out before depz_frame_destroy. data == NULL means that stream was disabled or not produced. shape is row-major: images are (H,W) or (H,W,C); a point cloud is (H,W,3).

Fields: data (first byte, or NULL if the stream is absent), dtype (DEPZ_U8 / DEPZ_F32), ndim (2 or 3), shape (extent per dimension, unused trailing dims 0), byte_size (total bytes behind data).
