Detailed Description

Core SDK-wide data structures and enumerations.

Provides lightweight plain-data types shared across the modular and high-level pipeline APIs. OpenCV types (cv::Mat) are used where hardware-accelerated processing is required; these structs cover cases where a dependency-free data holder is preferred.
