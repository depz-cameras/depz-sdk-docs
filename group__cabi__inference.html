Run a neural inference on one already-rectified stereo pair, without a pipeline.

The low-level primitive behind "disparity/depth from two images" — no camera, calibration, or worker thread. The two frames are resized to the model's input canvas, pushed through the network, cropped, and the map written to out_buf as row-major float32: disparity (canvas pixels) for a stereo net, depth (metres) for a mono net (right image ignored). left/right are tightly packed 8-bit images; channels is 1 (gray) or 3 (BGR, OpenCV order). Buffer units are float counts, not bytes: the map is written only when it fits (out_buf_floats >= *out_needed_floats); *out_width, *out_height, *out_needed_floats are always set. Probe by passing out_buf=NULL, out_buf_floats=0.

Returns
DEPZ_OK on success; DEPZ_ERR_INVALID_ARG when a non-NULL buffer is too small; DEPZ_ERR_INFERENCE when the network produced no float map.

Create a neural inference from an architecture slug, model file, and engine slug.

The C API does not download models — the caller resolves the file path (the Python registry handles downloads). arch_slug ∈ {lightstereo, crestereo-initial, crestereo-combined, depth-anything-v2-small, metric3d-v2-vit-small}; engine_slug ∈ {openvino-CPU, openvino-GPU, onnx-cpu, onnx-cuda}.

Parameters
[in]arch_slugNetwork architecture slug.
[in]model_pathFilesystem path to the resolved model file.
[in]engine_slugExecution engine slug ("{kind}-{device}").
[out]outReceives the new depz_inference handle (NULL on failure; free via depz_inference_destroy).
Returns
DEPZ_OK on success; nonzero depz_status on error.

Create a neural inference with a pinned NCHW input resolution.

Required for mono depth nets exported with dynamic input dimensions (metric3d-v2, depth-anything-v2), which otherwise fail to load with "Cannot get length of dynamic dimension". Pass the model's intended canvas size (e.g. 504×504 for metric3d-v2-vit-small). For stereo nets with static shapes this behaves like depz_inference_create. (C ABI v1.9+)

Parameters
[in]arch_slugNetwork architecture slug.
[in]model_pathFilesystem path to the resolved model file.
[in]engine_slugExecution engine slug ("{kind}-{device}").
[in]input_heightPinned NCHW input height in pixels.
[in]input_widthPinned NCHW input width in pixels.
[out]outReceives the new depz_inference handle (NULL on failure; free via depz_inference_destroy).
Returns
DEPZ_OK on success; nonzero depz_status on error.

Set the camera intrinsics (fx, fy, cx, cy) on a neural inference.

Typically a mono depth model (metric3d-v2 / depth-anything-v2) that needs intrinsics to recover metric scale; pixel units in the model's input canvas. No effect on stereo inferences that ignore intrinsics.

Returns
DEPZ_OK on success; nonzero depz_status on error.
