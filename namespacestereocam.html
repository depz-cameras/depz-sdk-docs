ParamValue stereocam::coerceParam ( const Control &  spec,
const ParamValue &  value 
)

Coerce + clamp + grid-snap value according to spec.kind.

Mirrors ControlOwner._coerce in _controls.py exactly:

  • "boolean": truthiness of value -> bool.
  • "enum": value must equal an allowed entry — string equality for string enums, numeric equality for numeric enums (an integral result is returned as int64_t).
  • "number": double, clamped to [min, max], snapped to the min + n*step grid, then returned as int64_t when the spec's default is integral.
Returns
The coerced value, ready to store as the parameter's current value.
Exceptions
std::invalid_argumentfor an unknown kind or an enum miss.

◆ depthToColoredPointCloud()

cv::Mat stereocam::depthToColoredPointCloud ( const cv::Mat &  disparity,
const cv::Mat &  Q,
const cv::Mat &  color_bgr 
)

Reproject a disparity map to a coloured 3-D point cloud (XYZRGB).

Combines the geometric reprojection (via Q) with per-pixel RGB colour sampled from the aligned rectified colour image. For each pixel the homogeneous reprojection \([X\ Y\ Z\ W]^\top = Q\,[u\ v\ d\ 1]^\top\) is divided by \(W\) and the BGR sample is normalised to \([0,1]\):

\[ \text{output}[u,v] = \Bigl(\tfrac{X}{W},\; \tfrac{Y}{W},\; \tfrac{Z}{W},\; \tfrac{R}{255},\; \tfrac{G}{255},\; \tfrac{B}{255}\Bigr) \]

Parameters
disparityInput disparity map (CV_32F).
Q4×4 reprojection matrix.
color_bgrRectified colour image aligned with disparity (CV_8UC3).
Returns
Dense XYZRGB point cloud (CV_32FC6).
  • XYZ in metres.
  • RGB normalised to \([0, 1]\).

◆ depthToPointCloud()

cv::Mat stereocam::depthToPointCloud ( const cv::Mat &  disparity,
const cv::Mat &  Q

Reproject a disparity map to a 3-D point cloud using the Q matrix.

Internally calls cv::reprojectImageTo3D() with the supplied Q matrix, producing one 3-D point per pixel:

\[ [X, Y, Z, W]^T = Q\,[u, v, d, 1]^T, \quad \mathbf{p} = (X/W,\;Y/W,\;Z/W) \]

Parameters
disparityInput disparity map (CV_32F).
Q4×4 reprojection matrix from StereoCalibration::getQMatrix().
Returns
Dense point cloud (CV_32FC3, millimetres).
Note
Divide result by 1000 to obtain metres.
cv::Mat Q = calib->getQMatrix();
cv::Mat pts = depthToPointCloud(disparity, Q); // mm
cv::Mat pts_m;
pts.convertTo(pts_m, CV_32F, 0.001); // m

double  fx,

double  cy

Depth map → organized point cloud via pinhole unprojection (metres in, metres out).

Inverse pinhole projection of every depth pixel \((u, v, z)\):

\[ \mathbf{p}(u,v) = \left( \frac{(u - c_x)\, z}{f_x},\; \frac{(v - c_y)\, z}{f_y},\; z \right) \]

Where depth is 0 the cloud entry is \((0, 0, 0)\) — consistent with the 0-for-invalid contract of disparityToDepthMeters().

◆ disparityToDepth()

cv::Mat stereocam::disparityToDepth ( const cv::Mat &  disparity,
double  baseline_mm,
double  focal_px 
)

Convert a disparity map to a depth map.

Applies the stereo geometry formula element-wise:

\[ Z[u,v] = \frac{f_x \cdot B}{d[u,v]} \]

Pixels with \(d \le \text{kMinDisparityPx}\) or with reconstructed depth outside \([\text{kMinDepthM},\,\text{kMaxDepthM}]\) are set to NaN (invalid / out-of-range). This invariant is required by downstream temporal filters and point-cloud builders — never use 0 as a sentinel.

Parameters
disparityInput disparity map (CV_32F, pixels).
baseline_mmStereo baseline \(B\) in millimetres (StereoCalibration::getBaseline()).
focal_pxFocal length \(f_x\) in pixels (getCameraMatrixLeft().at<double>(0,0)).
Returns
Depth map (CV_32F, millimetres). Same size as disparity.
double B = calib->getBaseline();
double fx = calib->getCameraMatrixLeft().at<double>(0, 0);
cv::Mat depth_mm = disparityToDepth(disparity, B, fx);
cv::Mat depth_m = depth_mm / 1000.0f;

Disparity → depth in metres, 0.0 for invalid pixels.

depz_camera_sdk contract: invalid pixels are 0.0 (not NaN), depth unit is metres, validity threshold is caller-supplied (default 0.1 px).

\[ Z_m(u,v) = \begin{cases} \dfrac{B \, f_x}{d(u,v)} & d(u,v) > \tau \\[6pt] 0 & \text{otherwise} \end{cases} \]

with baseline \(B\) in metres and disparity threshold \(\tau\).

◆ kMinDisparityPx

constexpr float stereocam::kMinDisparityPx = 0.5f
constexpr

Minimum disparity (px) below which depth is treated as invalid (NaN).

Derived from kMaxDepthM: d_min = focal_px * baseline_m / Z_max. 0.5px is conservative for typical baseline ~100mm and focal ~700px (≈140m max range).
