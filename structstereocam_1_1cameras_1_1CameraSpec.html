Detailed Description

Firmware-class description of a camera model.

Aggregates the capture modes (CaptureMode), the V4L2 / DirectShow controls (CameraControl), and the SKU strings the hardware reports. One CameraSpec corresponds to one firmware class (e.g. every Varius SKU shares the same modes and controls — only the per-unit calibration differs).
