Detailed Description

Abstract interface for stereo disparity inference backends.

All deep-learning stereo backends (OpenVINO, TensorRT, ONNX Runtime) implement this interface. The pipeline uses it through StereoDepth — never call backends directly from high-level pipeline code.

Supported backends:

  • OpenVINO — Intel CPU, GPU, and NPU acceleration
  • TensorRT — NVIDIA GPU (FP16/INT8 optimised)
  • ONNX Runtime — cross-platform CPU and CUDA
See also
StereoDepth, StereoInferenceFactory

Run stereo disparity inference on a rectified image pair.

Both images must be rectified (same resolution, epipolar-aligned). The disparity output is in pixel units (CV_32F).

Parameters
[in]leftLeft rectified image (CV_8UC1 or CV_8UC3).
[in]rightRight rectified image, same size as left.
[out]disparityOutput disparity map (CV_32F, pixels).
Returns
true on success.

Implemented in stereocam::TensorRTStereoInference, stereocam::OpenVINOStereoInference, and stereocam::ONNXRuntimeStereoInference.

Load an inference model from disk.

Accepted file extensions depend on the backend:

  • OpenVINO : .xml (IR) or .onnx
  • TensorRT : .engine or .onnx (auto-converts)
  • ONNX Runtime : .onnx
Parameters
model_pathPath to the model file.
deviceTarget device string, e.g. "CPU", "GPU", "AUTO", "CUDA". The interpretation is backend-specific.
Returns
true on success; the implementation must call LOG_ERROR before returning false.

Implemented in stereocam::TensorRTStereoInference, stereocam::ONNXRuntimeStereoInference, and stereocam::OpenVINOStereoInference.
