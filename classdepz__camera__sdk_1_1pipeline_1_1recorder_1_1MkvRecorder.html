Detailed Description

One-shot MKV recorder backed by the C++ libavformat adapter.

Construct → start() → feed() per frame → stop(). The writer thread
drains a bounded queue and calls into C++ for the actual mux —
keeps the camera pump thread free to grab the next frame at full
rate even when the disk lags.

is never called.
