Detailed Description

Aspect-preserving resize-and-pad preprocessing into an inference tensor.

Declares the easy_deploy::IImageProcessing interface and the CPU/CUDA factory functions. Each implementation resizes an input image to fit the model's input rectangle (preserving aspect), pads the remainder, optionally transposes HWC→CHW and normalises, and writes the result straight into the destination tensor.
