Detailed Description

Backend-agnostic ML inference engine interface (Adapter pattern).

Defines stereocam::engines::InferenceEngine, the single abstraction the pipeline and the Python InferenceEngine ABC see. Concrete implementations (OpenVINOEngineAdapter, OnnxEngineAdapter, TensorRTEngineAdapter) each wrap one third-party SDK in its own translation unit, so the rest of the code never includes <openvino/...> / <onnxruntime_cxx_api.h> directly. The contract mirrors depz_camera_sdk.engines.base.InferenceEngine.
