Detailed Description

Direct camera grab — pulls raw encoded frames straight from a source without a pipeline (no calibration / inference / depth).

Use for preview or recording the raw stream.

try (DepzSource src = DepzSource.camera("/dev/video0", 1280, 480, 20, "MJPG")) {


    OpenedInfo info = src.open();


    RawFrame f = src.grab(1.0);              // null on timeout


    BufferedImage img = ImageIO.read(new ByteArrayInputStream(f.jpeg()));


}
