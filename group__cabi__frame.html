Encode one stream of a frame as a display-ready JPEG.

8-bit streams (raw / rectified, BGR) are encoded as-is; float32 streams (disparity / depth) are normalised over the valid range and OpenCV colour-mapped first — the same visualisation the modular API's visualizeDisparity uses. colormap is an OpenCV COLORMAP_* value (-1 → JET); quality is 1..100 (0 → 90). 3D streams (point cloud) are rejected. *out_size always receives the full JPEG size; probe by passing out_buf=NULL, buf_size=0. (C ABI v2.3+)

Returns
DEPZ_OK on success; DEPZ_ERR_INVALID_ARG for a 3D stream or a too-small non-NULL buffer.
