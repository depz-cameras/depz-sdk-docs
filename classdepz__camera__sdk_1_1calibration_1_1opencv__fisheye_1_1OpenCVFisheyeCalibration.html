Detailed Description

Kannala–Brandt fisheye, lon/lat-rectified.

Python-side state mirrors the core runtime (`_cpp_fisheye_runtime`)
so existing consumers that read `self._R1` / `self._baseline_m` /
`self.image_size` keep working. Hot-path math is C++.
