Detailed Description

Native TensorRT inference engine (pre-compiled .engine files).

Compared to ``OnnxEngine("...", ep="tensorrt")`` (ORT TRT EP):
- No ORT layer — lower latency on hot paths.
- Loads instantly — no JIT compilation on first run.
- Engine file is GPU-specific; generate with ``trtexec``.

The synchronous ``compute()`` blocks until inference completes.
``submit()`` (from the ABC) fans it onto a ``ThreadPoolExecutor``
so the caller can pipeline frame preprocess with inference.
