DEPZ_API depz_status depz_fetch_camera_info_json ( const char *  serial,
char *  out_buf,
size_t  buf_size,
size_t *  out_needed 
)

Fetch the camera-info JSON for a serial number over HTTP.

Synchronous GET of https://<host>/functions/v1/camera-info?serial_number=<serial>, returning the response body as a NUL-terminated JSON string (mirrors the Python L0 helper fetch_camera_info_json). out_buf is always NUL-terminated; *out_needed receives the full payload size excluding NUL, so the caller can retry with a larger buffer. Pass buf_size=0 to just query the size.

Returns
DEPZ_OK on success; DEPZ_ERR_SOURCE on network failure (unreachable, non-2xx, malformed body).

DEPZ_API depz_status depz_list_engines ( char  out_slugs[][DEPZ_ENGINE_SLUG_MAX],
int32_t  max_count,
int32_t *  out_count 
)

Enumerate engines compiled into this build and available on this machine.

Slugs follow the "{kind}-{device}" format (L4) — e.g. "openvino-CPU", "onnx-cuda", "onnx-cpu — pre-sorted highest-priority first, so slugs[0] is the recommended default. out_slugs is a 2D buffer of [max_count][DEPZ_ENGINE_SLUG_MAX] bytes; pass max_count=0 to just query the count.

Returns
DEPZ_OK on success; nonzero depz_status on error.
