Parameters
arch_slugOne of lightstereo, crestereo-initial, crestereo-combined, depth-anything-v2-small, metric3d-v2-vit-small.
model_pathAbsolute path to the .onnx / .xml file (caller resolves; no download happens here).
engine_slugOne of openvino-CPU, openvino-GPU, onnx-cpu, onnx-cuda, tensorrt-cuda.
input_hFixed model input height; 0 for nets whose ONNX already declares static dims (stereo). Required for dynamic single-input nets (depth-anything).
input_wFixed model input width; see input_h.
output_nameNamed model output for multi-head nets; "" = default.

Ready inference object.
