Detailed Description

Pinhole (OpenCV5/8) stereo-or-mono calibration with canvas-fit remap.

Stereo rectification is delegated to an embedded StereoCalibration; compute_canvas_remap() rescales the rectified projection into the model input canvas before building the cv::remap tables.
