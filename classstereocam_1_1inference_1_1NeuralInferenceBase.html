Detailed Description

Shared partial implementation of DepthInferenceBase.

Handles the bookkeeping every network shares — engine pointer, slug strings, intrinsics storage, input_size/label/warmup and the sync-submit glue — leaving subclasses to override only preprocess and postprocess. Holds a shared_ptr<InferenceEngine>, so the engine outlives the inference even if the pipeline rebuilds the inference object mid-stream.

Provide canvas-pixel intrinsics to focus-aware mono networks.

No-op for stereo and intrinsic-free mono networks; Metric3D V2 / UniDepth override it. The pipeline recomputes fx … cy from the active calibration on every model-resolution change.
