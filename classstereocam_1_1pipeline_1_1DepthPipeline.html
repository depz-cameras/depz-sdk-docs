Detailed Description

C++ worker loop for the depz_camera_sdk: source → rectify → inference → filters → queue.

Drop-in replacement for the Python DepthPipeline orchestrator. Owns one worker thread (started in start(), joined in stop()) that drives the full chain — grab, decode, split side-by-side, rectify, neural inference, FrameProcessor, and FilterChain — and pushes each result as a FramePacket into a drop-oldest queue. For a seekable file source it also exposes replay controls (pause / resume / seek / step).

Single-shot lifecycle: create → start() → get() loop → stop() → discard.

See also
StereoPipeline — the higher-level istereolab orchestration API.
