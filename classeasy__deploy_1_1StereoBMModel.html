Detailed Description

StereoBM implementation using OpenCV cv::StereoBM.

This class provides a CPU-based stereo matching algorithm using OpenCV's Block Matching (BM) algorithm. It's a simple, standalone implementation without async pipeline dependencies.

Principles: SOLID, Single Responsibility, Strategy Pattern
