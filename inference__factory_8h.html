Detailed Description

Single factory turning (arch slug, model file, engine slug) into a ready DepthInferenceBase.

The one source of truth for the slug → engine + architecture wiring; replaces the duplicated mapping that lived in both the C API (depz_c_api.cpp) and the Python build_cpp_inference (cpp_adapter.py).
