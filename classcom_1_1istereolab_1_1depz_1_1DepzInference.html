Detailed Description

A standalone neural inference — the small "two images → disparity map" primitive, with no camera, no calibration, and no pipeline.

Build it from an architecture slug + model file + engine, then call disparity(byte[], int, int, byte[], int, int, int) on a pre-rectified stereo pair.

try (DepzInference inf = DepzInference.create("lightstereo", modelPath, null)) {


    DepzImage disp = inf.disparity(leftBgr, w, h, rightBgr, w, h, 3);


    float[] map = disp.floats();   // disparity in canvas pixels


}

For a stereo network the map is disparity (canvas px); for a mono depth network it is depth in metres and the right image is ignored.
