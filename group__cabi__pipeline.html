◆ depz_pipeline_set_stream_enabled_graph()

DEPZ_API depz_status depz_pipeline_set_stream_enabled_graph ( depz_pipeline *  ,
depz_stream  stream,
int32_t  enabled,
int32_t  cascade,
int32_t *  out_applied,
char *  out_reason,
size_t  reason_buf_size,
depz_stream *  out_cascaded_streams,
int32_t *  out_cascaded_count 
)

Toggle a stream, cascading to dependent streams when requested.

Like depz_pipeline_set_stream_enabled, but when cascade != 0 and the change would orphan a downstream stream (e.g. disabling depth while point_cloud is on), the orphaned streams are toggled too.

Parameters
[in]hPipeline handle.
[in]streamWhich stream to toggle.
[in]enabledDesired flag value (0 = off, non-zero = on).
[in]cascadeWhen non-zero, also toggle dependent streams.
[out]out_appliedReceives the flag value actually applied.
[out]out_reasonCaller-supplied buffer for a human-readable reason; pass at least 128 bytes.
[in]reason_buf_sizeCapacity of out_reason in bytes.
[out]out_cascaded_streamsCascaded depz_stream values (at most DEPZ_STREAM_COUNT entries); NULL to skip.
[out]out_cascaded_countNumber of cascaded streams written; NULL to skip.
Returns
DEPZ_OK on success; nonzero depz_status on error.
