InferenceFuture depz_camera_sdk.inference.native_classical.NativeClassicalInferenceBase.submit (   self,
np.ndarray  canvas_l,
np.ndarray  canvas_r

Submit one stereo pair via the native ``submit_resized_sync``
C++ entry point (resize happens inside the kernel — no Python
cv2.resize on the hot path). All three built-in classical
backends (SGBM, StereoBM, RectifyOnly) expose
``submit_resized_sync``; the W8 cleanup retired the in-Python
cv2.resize fallback because no production concrete needs it.

Reimplemented from depz_camera_sdk.inference.base.DepthInferenceBase.
