iStereoLab / DEPZ SDK — API Documentation

A C++17 stereo-vision SDK: camera → rectify → disparity (neural or classical) → metric depth → point cloud, with cross-language bindings (Python, Java, Node, ctypes) over one core. This is the public API reference, generated from the public headers; the implementation lives in the private SDK repository.

Core math

Disparity → metric depth:

\[ Z = \frac{f\,B}{d} \]

Back-projection of \((u,v,Z)\) to a camera-frame point:

\[ X = \frac{(u - c_x)\,Z}{f_x}, \qquad Y = \frac{(v - c_y)\,Z}{f_y} \]
