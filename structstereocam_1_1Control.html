Detailed Description

Pure-C++ descriptor of one runtime-tunable parameter.

Core counterpart of the Python Control dataclass (_controls.py). The SDK core declares its tunable surface — filter/inference/calibration controls — as std::vector<Control>, with no dependency on nanobind or Python. The binding layer projects each Control onto a Python Control (and converts values back) at the _core boundary.

Field order and semantics mirror the Python dataclass so the binding projection is a field-for-field copy.
