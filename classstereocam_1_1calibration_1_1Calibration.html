Detailed Description

Stereo / mono calibration runtime contract.

Every concrete impl owns its own intrinsics + extrinsics representation and exposes the methods needed by the SDK pipeline. Mono calibrations report is_stereo() == false; the right-eye half of rectify_pair / canvas_remap_pair returns an empty cv::Mat and baseline_m() is 0.

Implementations must call compute_canvas_remap() before canvas_remap_pair(); querying canvas geometry (focal_canvas_px, canvas_layout, …) before that point returns the default state (has_canvas_remap == false).
