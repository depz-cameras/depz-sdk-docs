Detailed Description

Toolkit-compatible cloud builder for Python C-ABI users.

It consumes filtered or raw disparity, applies optional spatial denoise,
per-ray Z accumulation, and temporal disparity-variance gating, then
unprojects through ``DepzCalibration``. This mirrors depz-toolkit's
``disp_cloud`` data path without requiring the toolkit process.
