Detailed Description

Чистый C++ класс для GStreamer камеры (без Qt зависимостей)

Принцип единой ответственности:

  • Только захват кадров из GStreamer pipeline
  • Прецизионный замер FPS
  • Управление pipeline (init/start/stop)

НЕ ДЕЛАЕТ:

  • Qt сигналы/слоты (это задача обёртки)
  • Рисование (это задача рендера)
  • Обработка кадров (это задача процессора)
