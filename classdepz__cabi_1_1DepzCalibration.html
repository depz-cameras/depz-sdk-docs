Detailed Description

The stereo geometry: intrinsics, distortion, the left↔right transform and the baseline.

It turns a raw pair into a rectified pair and disparity into metric depth. Load it with :meth:from_file / :meth:from_json / :meth:from_web.

Unproject a depth map to a 3D cloud (client-side numpy, not a C call).

Each pixel \((x,y)\) with depth \(Z\) back-projects through the pinhole model to camera-frame coordinates:

\[ X = \frac{(x - c_x)\,Z}{f_x}, \quad Y = \frac{(y - c_y)\,Z}{f_y}, \quad Z = Z \]

Parameters
depthHxW float depth map (m).
fx,fyFocal lengths (px); default to the canvas focal.
cx,cyPrincipal point (px); default to the image centre.
Returns
HxWx3 float32 cloud (X, Y, Z) in metres; \(Z\le0\) → (0,0,0).

Project disparity to metric depth (client-side numpy, not a C call).

Each pixel's depth follows from the stereo triangle:

\[ Z = \frac{f \cdot B}{d} \]

where \(f\) is the canvas focal length (px), \(B\) the baseline (m) and \(d\) the disparity (px). Both \(f\) and \(B\) come from this calibration.

Parameters
disparityHxW float disparity map (px).
valid_thresholdPixels with \(d \le\) this become 0 m.
Returns
HxW float32 depth in metres; invalid/non-positive \(d\) → 0.0, matching the SDK depth-stream contract.

Undistort + stereo-rectify a native-resolution pair so that corresponding points share a scanline (epipolar lines become rows):

\[ \tilde{p} = K\, R\, K^{-1}\, \mathrm{undistort}(p) \]

with \(K\) the intrinsics and \(R\) the per-eye rectifying rotation.

Call prepare() first.

Parameters
left,rightNative-resolution raw eyes (HxW[xC] uint8).
Returns
(left_rect, right_rect) — equal-size uint8 arrays on the canvas.
