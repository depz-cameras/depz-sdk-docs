Detailed Description

Thin compatibility classes for neural inference registry entries.

The public registry still exposes class objects for
``available_inferences()`` / ``get_inference()``, but the runtime
construction path goes through :func:`build_cpp_inference` and
returns :class:`CppInferenceAdapter`. These classes preserve the
metadata contract while sourcing neural metadata from the native
nanobind catalog whenever it is available.
