inline

Rectify a raw interleaved pair onto the prepared canvas.

Each eye is warped by its rectifying homography \( K\,R\,K^{-1} \) so a world point lands on the same row in both. The output size is queried by a probe call, then filled.

Parameters
[in]leftLeft image bytes (row-major, interleaved channels).
[in]rightRight image bytes (same layout).
[in]widthInput width per eye.
[in]heightInput height per eye.
[in]channelsChannels per pixel (3 = BGR, 1 = gray).
Returns
The rectified pair and its canvas dimensions.
