Detailed Description

A recorded .mkv presented as a seekable FrameSource for replay.

Wraps an IMkvSource (libavformat demuxer) so the C++ worker pipeline consumes a recording through the same grab-contract as a live camera. Being seekable and EOF-reporting, it puts DepthPipeline into replay mode (pause / step / seek). Container metadata harvested at open() lets the Python shim rebuild the recording's CameraInfo / calibration.
