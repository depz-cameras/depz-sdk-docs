◆ depz_status

enum depz_status

Result code returned by almost every C ABI call.

DEPZ_OK (0) is success; any nonzero value is a failure — call depz_last_error() for a human-readable reason. The C ABI throws no exceptions.

Enumerator
DEPZ_OK 

Success.

DEPZ_ERR_INVALID_ARG 

NULL handle or otherwise bad parameter.

DEPZ_ERR_NOT_RUNNING 

Pipeline used before start().

DEPZ_ERR_ALREADY_RUNNING 

start() called on an already-running pipeline.

DEPZ_ERR_TIMEOUT 

get() produced no frame within the timeout.

DEPZ_ERR_SOURCE 

Camera open / frame-grab failure.

DEPZ_ERR_INFERENCE 

Model load / compute failure.

DEPZ_ERR_CALIBRATION 

Calibration missing or could not parse.

DEPZ_ERR_UNKNOWN_STREAM 

Requested a stream the frame does not carry.

DEPZ_ERR_UNKNOWN_PARAM 

Unknown control / parameter name.

DEPZ_ERR_INTERNAL 

Unexpected internal error.
