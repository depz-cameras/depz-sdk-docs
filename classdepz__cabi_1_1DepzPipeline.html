Detailed Description

The running worker: camera/file -> rectify -> inference -> filters -> output streams.

Build it, :meth:start, pull frames with :meth:get, then :meth:close.

Tune one filter live, without restarting the pipeline.

Example — the temporal EMA filter smooths each pixel over time:

\[ \mathrm{out}_t = \alpha\,\mathrm{new}_t + (1-\alpha)\,\mathrm{out}_{t-1} \]

so set_filter_param("temporal_ema", "alpha", 0.2) sets \(\alpha\).

Parameters
slugFilter name (e.g. "temporal_ema", "speckle").
keyParameter name within that filter.
valuebool / int / float — routed to the matching C setter by type.
