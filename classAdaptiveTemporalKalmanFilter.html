Detailed Description

Per-pixel adaptive Kalman filter for depth / disparity map stabilisation.

A scalar (1-D) Kalman filter is run independently at every pixel \((u,v)\), tracking the depth (or disparity) value over time. The model is a random walk (static transition, \(F = 1\)): the depth is assumed constant between frames, with all motion absorbed by the process noise \(Q\). State per pixel is the estimate \(\hat{x}\) (m_estimate) and its error covariance \(P\) (m_covariance).

Predict step

The estimate carries over unchanged; only the covariance grows by the process noise:

\[ \hat{x}_{k|k-1} = \hat{x}_{k-1|k-1}, \qquad P_{k|k-1} = P_{k-1|k-1} + Q' \]

where \(Q' = Q \cdot m_\mathrm{imu}\) is the IMU-scaled process noise (adaptiveProcessNoise = m_processNoise * m_imuMotionFactor). Faster camera motion inflates \(Q'\), letting the estimate track quicker.

Adaptive measurement noise

The measurement noise \(R[u,v]\) (m_adaptiveR) is raised wherever the new frame departs from the current estimate, so abrupt depth changes are trusted less and brief transients are smoothed:

\[ g[u,v] = \min\!\left(\frac{\bigl|z_k[u,v] - \hat{x}[u,v]\bigr|}{20},\; 1\right), \qquad R[u,v] = R_{\text{base}} + s \cdot g[u,v] \]

Here \(z_k\) is the incoming depth, \(R_{\text{base}}\) is m_measurementNoise, \(s\) is m_adaptationStrength, and \(g \in [0,1]\) is the normalised temporal change (cv::absdiff against m_estimate, scaled by 20 and clamped). This is a temporal-difference adaptation, not a spatial Sobel gradient.

Update step

\[ K_k = \frac{P_{k|k-1}}{P_{k|k-1} + R[u,v]} \]

\[ \hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k \bigl(z_k - \hat{x}_{k|k-1}\bigr) \]

\[ P_{k|k} = (1 - K_k)\, P_{k|k-1} \]

The covariance is then clamped to \([10^{-3},\,10]\). Pixels whose measurement is invalid ( \(z_k \le\) m_invalidDepthThreshold) skip the update — the estimate is held and \(P\) is gently re-inflated to \(1.5\) so the filter re-locks quickly when valid data returns.

All operations run in-place on pre-allocated cv::Mat buffers for real-time performance (≥ 30 FPS at 640×480).

Run one predict/update pass of the per-pixel Kalman filter.

Applies, in order: adaptive \(R\) from the temporal difference \(|z_k - \hat{x}|\), the predict \(P \mathrel{+}= Q'\), the gain \(K = P/(P+R)\), the state update \(\hat{x} \mathrel{+}= K\,(z_k - \hat{x})\), and the covariance update \(P \leftarrow (1-K)\,P\). On the first frame (or a size change) the state is seeded from the input and returned unchanged.

Parameters
[in]inputDepth or disparity map; non-CV_32F is converted internally.
[out]outputFiltered estimate, same size and type as input.
Returns
true always; disabled or empty input passes through unfiltered.

Feed IMU motion magnitudes to scale the process noise \(Q\).

Combines the three magnitudes into a single motion factor \(m_\mathrm{imu} = \mathrm{clamp}(0.5\,\|\omega\| + 0.3\,\|a\| + 0.2\,\Delta q,\,0,\,2)\), which multiplies \(Q\) in the predict step. Larger motion ⇒ larger \(Q' = Q\,m_\mathrm{imu}\) ⇒ faster tracking through camera movement.

Parameters
[in]gyroMagnitudeAngular-rate magnitude \(\|\omega\|\).
[in]accelMagnitudeLinear-acceleration magnitude \(\|a\|\).
[in]quaternionChangeOrientation-change magnitude \(\Delta q\).
