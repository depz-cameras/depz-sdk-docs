Detailed Description

SDK stream-style logger and the LOG_* macros.

The canonical logging facility for the SDK — use LOG_DEBUG, LOG_INFO, LOG_WARNING, LOG_ERROR with << stream syntax. Do not use std::cerr or the legacy log.hpp.
