Run the postprocess chain on one raw inference output.

Stereo path (inference->outputs_depth() == false):

  1. inference.postprocess(raw, layout) → disparity map
  2. if (depth || cloud): disparity_to_depth(disp, baseline_m, focal_px)
  3. if cloud: depth_to_point_cloud(depth, fx, fy, cx, cy)

Mono path (outputs_depth == true):

  1. inference.postprocess(raw, layout) → depth map
  2. if cloud: depth_to_point_cloud(depth, fx, fy, cx, cy)

Result fields are empty when their stream flag is false; disparity is always empty on the mono path.
