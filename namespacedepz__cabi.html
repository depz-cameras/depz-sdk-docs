Detailed Description

depz_cabi — Python binding over the flat C ABI (depz_c_api.h), via ctypes.

This is the SAME functor as the Java FFM binding (`com.istereolab.depz`): both
factor through the one C ABI, so their observables are equal *by construction*
(zero cross-language entropy), not merely within tolerance. It deliberately
mirrors the Java wrapper's surface (Depz, DepzCalibration, DepzInference,
DepzPipeline, DepzFrame, DepzImage, Stream) so the examples read 1:1.

Contrast: the production `istereolab` is nanobind over the C++ classes — a
different presentation of the same semantics (naturally isomorphic only).

Lib resolution: $DEPZ_LIBRARY, else <repo>/build/libdepz_istereolab.so.
