Detailed Description

Concrete :class:`DepthInferenceBase` subclasses.

Thin runtime wrappers are generated dynamically here when a per-model
module only carries metadata over an already-native implementation.
This lets us delete one-class-per-file stubs while preserving the public
class names and direct submodule import paths.
