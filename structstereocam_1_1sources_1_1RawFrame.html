Detailed Description

A single grabbed frame in source-native wire format (typically JPEG).

Zero-copy contract — data points into a backend-owned buffer (e.g. a GstMapInfo into a GstBuffer of an appsink sample). The buffer stays valid until release() is called; the caller MUST call release() once it has consumed the bytes (typically right after building a Python bytes object).
