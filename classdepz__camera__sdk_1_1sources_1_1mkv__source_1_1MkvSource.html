Detailed Description

Frame source for replaying an MKV recorded by :class:`MkvRecorder`.

``replay_mode`` ∈ {"realtime", "every_frame"}:
  - ``"realtime"`` (default): pump thread paces grabs to PTS;
    ``grab()`` returns the latest frame.
  - ``"every_frame"``: synchronous, no pump; ``grab()`` returns
    the next frame as fast as the caller pulls. Used by the
    recorder→replay round-trip tests + offline batch jobs.

◆ list_camera_controls()

list depz_camera_sdk.sources.mkv_source.MkvSource.list_camera_controls (   self) 
Snapshot of the camera-property controls available on this
source. Returns ``[]`` for sources that don't expose controls
(replay, mocks). Live cameras override.

Each :class:`CameraControl` carries a *snapshot* of `current_value`
/ `min` / `max` / `step` at the moment of the call — these may
change over time (auto-exposure, manual writes). Re-call to
refresh.

Controls not actually available on the current platform are
omitted. The returned list is therefore "what you can really
use right now" — safe to drive a UI from directly.
