Detailed Description

ONNX Runtime implementation of the legacy IStereoInference 2-input API.

Two-input stereo backend over Ort::Session, with cached planar (CHW) preprocessing buffers reused across frames. Execution-provider selectable (CPU, CUDA, TensorRT EP, DirectML, CoreML). Serves the older StereoDepth path; the modern multi-input path uses engines::OnnxEngineAdapter.
