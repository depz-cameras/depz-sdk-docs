Detailed Description

A loaded neural inference — the standalone form of what a Pipeline::Builder builds internally.

Mirrors the Java DepzInference. The Pipeline::Builder takes engine/model slugs directly; this class is for code that wants to own an inference handle on its own (inspection, reuse, direct interop).

Run the network on a raw BGR pair (no pipeline).

The same operation the Java inf.disparity(...) and Python inf.disparity(...) expose — a stereo net returns disparity (canvas px), a mono net returns depth. Sizes itself via a probe call.

Parameters
[in]leftLeft image bytes (row-major, interleaved channels).
[in]rightRight image bytes (same layout; pass left again for mono).
[in]widthImage width per eye.
[in]heightImage height per eye.
[in]channelsChannels per pixel (3 = BGR).
Returns
The float map and its dimensions.
