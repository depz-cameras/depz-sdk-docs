Detailed Description

Stereo camera calibration: load, validate, rectify.

StereoCalibration is the concrete implementation of IStereoCalibrationModel. It parses a JSON calibration file (custom built-in parser, no cv::FileStorage dependency), computes the full rectification maps, and exposes all relevant intrinsic and extrinsic matrices.

Camera model

Each camera is described by the pinhole intrinsic matrix

\[ K = \begin{bmatrix} f_x & 0 & c_x \\ 0 & f_y & c_y \\ 0 & 0 & 1 \end{bmatrix} \]

and the distortion coefficient vector

\[ \mathbf{d} = (k_1,\, k_2,\, p_1,\, p_2,\, k_3,\, k_4,\, k_5,\, k_6) \]

following the rational distortion model used by OpenCV.

JSON structure

The loader expects the following nested layout:

{ "calibration_data": { "calibration_data": { "calibration": {
"camera_matrix_left": [[...], ...],
"camera_matrix_right": [[...], ...],
"distortion_coefficients_left": [[...]],
"distortion_coefficients_right": [[...]],
"rotation_matrix": [[...], ...],
"translation_vector": [[...]],
"image_width": 640,
"image_height": 480
}}}}
