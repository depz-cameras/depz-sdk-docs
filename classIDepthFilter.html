Detailed Description

Pure-virtual interface for a single-stage depth / disparity filter.

Each filter does exactly one thing (SRP):

  • Temporal smoothing via Kalman (AdaptiveTemporalKalmanFilter)
  • Spatial smoothing (bilateral, median — custom)
  • Motion compensation (optical flow — custom)

The pipeline calls apply() once per frame between the disparity inference step and the depth conversion step. reset() is called when the sequence restarts (e.g. after StereoPipeline::stop() / start()).
