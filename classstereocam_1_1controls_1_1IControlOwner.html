Detailed Description

Language-neutral base class for objects with self-describing runtime parameters.

Subclasses override parameters() to return the list of control descriptors. The host language's binding layer typically also provides control_get / control_set / controls_to_dict orchestration on top — those are stateful and necessarily host-side (they reach into the host's attribute storage).

Return this owner's full set of parameter descriptors.

Typically a fixed list per concrete class; if it varies (rare), the host orchestration layer recomputes per call. Override is pure-virtual — every concrete owner declares its surface.
