Open an ONNX model and select an execution provider.

Parameters
model_pathPath to the .onnx file.
providerORT EP name ("CPUExecutionProvider", "CUDAExecutionProvider", "TensorrtExecutionProvider") or "" for auto (try CUDA, fall back to CPU).
input_shapeOptional NCHW override for a single dynamic-shape input.
output_nameOptional named output to read; "" selects the first.
num_threads>0: strict CPU mode with N intra-op threads; 0: auto (tries CUDA EP, falls back to CPU).
