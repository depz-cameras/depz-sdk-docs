Detailed Description

Camera-property control descriptor (V4L2 / DirectShow control).

Pure-C++ extension of stereocam::Control (parameter description) with four camera-specific fields used by the V4L2 / DirectShow backends. The Python CameraControl dataclass is a projection of this struct — the binding layer copies field-for-field.

Field order mirrors the Python @dataclass(frozen=True, slots=True) CameraControl(Control) so positional construction maps 1:1.
