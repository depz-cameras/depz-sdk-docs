const CameraSpec& stereocam::cameras::specs::depz_varius ( )

Built-in DEPZ camera-spec data tables (formerly depz_camera_sdk.cameras.depz_*.py).

Each accessor returns a reference to a Meyers-singleton CameraSpec initialised on first use. The data is byte-equivalent to the Python reference (parity test: tests/test_camera_specs_parity.py).

Minima / Clarity / Mono / MonoMini share Varius's modes and controls — only their skus and (for Mono) the per-mode stereo flag differ.
