Detailed Description

Stage-agnostic filter base class.

Concrete filters override apply() to mutate out in-place (callees may resize / retype it via cv::Mat::create) and return true on success. The default reset() is a no-op — temporal filters (e.g. TemporalEmaFilter) override it to drop their prev-frame cache.
