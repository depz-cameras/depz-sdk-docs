Detailed Description

Intel OpenVINO backend implementing InferenceEngine.

Wraps ov::Core / ov::CompiledModel behind the engine interface and adds an N-slot ov::InferRequest pool so synchronous compute() and submit_async() can run concurrently. Targets CPU, Intel GPU, and NPU.
