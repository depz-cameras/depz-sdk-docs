Detailed Description

Monocular-depth marker class. The runtime mono path goes through
`NativeMonoInferenceRuntime` (multi-inherits from this) and
`CppInferenceAdapter`, which owns depth_scale/offset controls natively;
this class only contributes `OUTPUTS_DEPTH = True` to the MRO.
