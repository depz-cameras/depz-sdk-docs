Detailed Description

The stereo geometry: intrinsics, distortion, the left↔right transform and the baseline.

It turns a raw pair into a rectified pair. Load it with fromFile / fromJson / fromWeb.

◆ rectify()

depz::DepzCalibration::rectify (   options)
inline

Undistort + stereo-rectify a native-resolution pair so that corresponding points share a scanline (epipolar lines become rows):

\[ \tilde{p} = K\, R\, K^{-1}\, \mathrm{undistort}(p) \]

with \(K\) the intrinsics and \(R\) the per-eye rectifying rotation. Call prepare first.

Parameters
optionsThe raw left/right buffers and their width/height/channels.
Returns
The rectified pair { left, right, width, height, channels }.
