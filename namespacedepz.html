Detailed Description

Root namespace of the C++ C-ABI wrapper.

The DEPZ Node.js binding — the OOP face of the SDK, in TypeScript.

These Depz* classes (DepzPipeline, DepzCalibration, DepzFrame, …) are the handle-as-object form of the SDK: each wraps an opaque handle from the flat C ABI (depz_c_api.h) — not the C++ core directly. The native N-API addon calls the same 70 C functions as the Java (Panama FFM) and Python (ctypes) bindings, so Node's numbers are byte-identical to theirs by construction.

Layering: depz::Depz* (this TS facade) -> N-API addon -> flat C ABI -> C++ core.
