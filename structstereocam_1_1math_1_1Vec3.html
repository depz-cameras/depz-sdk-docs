Detailed Description

Immutable-style 3-D vector of float components \((x, y, z)\).

Supports the standard Euclidean vector algebra:

  • Addition / subtraction
  • Scalar multiplication / division
  • Dot product \(\mathbf{a} \cdot \mathbf{b} = a_x b_x + a_y b_y + a_z b_z\)
  • Cross product \(\mathbf{a} \times \mathbf{b}\)
  • Euclidean length \(\|\mathbf{v}\| = \sqrt{x^2 + y^2 + z^2}\)
